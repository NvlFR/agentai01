# VISION.md

## Misi

Membangun platform runtime tempat satu perusahaan bisa dijalankan oleh jaringan agent AI yang terstruktur, dapat diaudit, dan bisa dioperasikan lewat web, messaging, dan terminal.

## Arah Produk

AgentAI01 bukan sekadar chatbot tunggal. Targetnya adalah:
- organisasi agent bertingkat
- pembagian kerja per departemen
- orchestration yang jelas
- operator controls yang aman
- tool access yang sempit dan terukur

## Pilar Saat Ini

1. Runtime contracts yang ketat
2. Hierarchy registry untuk heads dan specialists
3. Provider-backed specialist execution
4. MCP integration yang realistis
5. Operator surfaces: API, Telegram, TUI

## Near-Term Priorities

- memperkaya workflow operator
- memperjelas docs adaptasi `restored-src` dan `openclaw`
- mengaktifkan MCP yang benar-benar dipakai
- memperkuat agent management dan observability

## Design Bias

- prefer composable systems
- prefer contract yang jelas daripada magic
- prefer gradual adaptation daripada big-bang port
