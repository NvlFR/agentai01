# Adaptation Baseline Principles

Prinsip utama adaptasi referensi eksternal ke `agentai01`:

- referensi dipakai sebagai sumber pola, bukan target copy-paste
- contract aktif repo ini tetap sumber kebenaran
- behavior harus dipetakan ke boundary runtime yang sekarang
- translasi dilakukan bertahap, bukan big-bang port
- naming publik harus netral project, jangan membawa brand referensi tanpa alasan

## Target yang Dijaga

- runtime contracts tetap stabil
- security dan secret flow tetap aman
- operator surfaces tetap bisa dipakai
- docs bisa menjelaskan apa yang diadopsi dan apa yang sengaja tidak diadopsi
