# Low Priority Adaptation Surface

Area ini optional dan bukan prioritas inti runtime:

## Examples

- premium TTS
- image/video generation enrichment
- advanced search providers
- shell/phone control surfaces default-off
- QA lab extensions

## Rule

- aktifkan hanya bila ada use case nyata
- jangan naikkan risk profile repo tanpa operator need yang jelas
- semua secret tetap dari env
- semua surface berisiko harus punya audit-friendly behavior
