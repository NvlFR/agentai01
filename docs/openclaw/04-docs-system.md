# OpenClaw Docs System

OpenClaw punya docs tree besar dan matang, dengan:

- docs index
- concept docs
- channel docs
- gateway docs
- automation docs
- help/troubleshooting docs

## Pelajaran untuk AgentAI01

- docs sebaiknya dipecah per topik
- index docs harus ringan
- reference docs dan adaptation docs sebaiknya dipisah
- file pengarah lama boleh dipertahankan sebagai redirect manual
