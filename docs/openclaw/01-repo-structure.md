# OpenClaw Repo Structure

`referensi/openclaw/` adalah monorepo besar yang mencampur:

- runtime TypeScript
- extensions/plugins
- apps native Apple dan Android
- docs system
- deployment/build scripts

## Struktur Besar

- `apps/`
- `docs/`
- `extensions/`
- `src/`
- `config/`
- `deploy/`
- `.github/`

Untuk adaptasi ke `agentai01`, yang paling relevan biasanya `src/`, `extensions/`, dan beberapa policy/docs file.
